# .qwen

Ce dossier contient les fichiers de configuration utilisés par Qwen lors des interventions précédentes sur le repo.

## Fichiers

- `settings.json`
- `settings.json.orig`

## Rôle

Ces fichiers ne participent pas au runtime de l’extension dans VS Code. Ils servent surtout de trace de configuration d’outillage externe.

## Installation

Aucune installation requise. Ce dossier peut être conservé pour audit ou retiré du package si vous ne souhaitez pas embarquer de configuration liée à Qwen.
